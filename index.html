About Chronoscope

Read the traces.

Chronoscope is a historical map game about reading the traces left inside old images. Each round asks a simple question: where does this image show, and when?

Observation

The answer lies not in memorization, but in observation: architecture, clothing, roads, signs, terrain, technology, and the quiet details of daily life.

Public History

The project is built as an experiment in public history, visual reasoning, and geographical imagination. Submissions require verification before they can enter the game.
